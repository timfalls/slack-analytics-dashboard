# slack-analytics-dashboard
Slack channel analytics dashboard for #build-with-winslow
Latest version:
  Upload Members CSV and Channel Stats CSV from Slack Analytics (drag and drop supported)
  Auto-parses what it can; you fill in the rest manually
  Remembers last week's total member count so it auto-calculates delta next week
  Generates the formatted Slack update for Niel and the tab-separated tracker row
  Keeps a running history of previous weeks with member counts
